# Weather App

## Сборка и запуск проекта

### 1. Клонирование репозитория

```bash
git clone https://github.com/Nurdoolota/weather-app.git
cd weather
```

### 2. Установка зависимостей

```bash
npm install
```

### 3. Запуск проекта

##### Android:
```bash
npm run android
```

### 4. Создание apk файла
Перейти в папку android:
```bash
cd android
```

выполнить команду
```bash
gradlew.bat assembleRelease
```
"
в директории "android\app\build\outputs\apk\release" создастся apk файл коротую можно запускать непосредственно в самом android устройстве

# План монетизации и публикации приложения

## 1. Монетизация

### Бесплатная версия с рекламой
- Основной функционал доступен бесплатно.
- В приложении интегрируются нативные баннерные и межстраничные рекламные модули.
- Пользователь периодически видит ненавязчивую рекламу при переходе между экранами.

### Премиум версия (Pro)
- Отключение рекламы через разовую покупку или по подписке.
- Возможность добавлять больше городов в избранное.
- Эксклюзивные функции: расширенный прогноз (на 7–10 дней), виджеты для рабочего стола, индивидуальные уведомления о погоде.
- Покупка осуществляется через Google Play Billing (Android).

### Партнёрские программы
- Интеграция с туристическими сервисами.

# Безопасность приложения

## 1. Хранение пользовательских данных
- Данные о городах, настройках и истории поиска хранятся **только локально на устройстве пользователя** (AsyncStorage/Keychain/Secure Storage).
- Не используются внешние серверы для хранения персональных данных.

## 2. Работа с API и сетевыми запросами
- Все сетевые запросы к погодному API осуществляются по защищённому протоколу HTTPS.
- Ключи API хранятся вне исходного кода.

## 3. Защита от взлома и модификации
- Приложение собирается с включённой ProGuard/Minify (Android) для усложнения декомпиляции.
- Проверяется целостность приложения при запуске .

## 4. Защита платежей и подписок
- Для покупок используются только официальные системы Google Play.
- Вся логика проверки покупок реализована через защищённые SDK платформы, а данные о покупках сверяются с сервером Google.

## 5. Приватность
- Приложение не собирает личные данные пользователя (имя, номер телефона и пр.).
- При необходимости работы с геолокацией пользователь явно даёт разрешение на её использование и может отключить доступ в любой момент.
